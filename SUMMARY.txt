===================================================================
BLUR EFFECTS EXPLANATION
===================================================================

BlurView creates the signature Apple glass effect:
- Logo: Heavy blur (80) for prominent glass
- Cards: Medium blur (60) for subtle glass
- Icons: Heavy blur (80) with extraLight tint

Layering technique:
1. Background color with opacity
2. BlurView with medium intensity
3. Inner view with lower opacity
4. Content on top

This creates depth and that "liquid glass" feel.

===================================================================
BEST PRACTICES
===================================================================

Performance:
- Use useNativeDriver: true for animations
- Limit BlurView nesting (max 2-3 levels)
- Avoid animating layout properties
- Use platform-specific shadows/elevation

Styling:
- Consistent border radius (18, 24, 28pt)
- Spacing multiples of 4 or 8
- High contrast for accessibility
- Platform.select for platform code

Accessibility:
- Text contrast meets WCAG standards
- Touch targets minimum 44x44pt
- Semantic font sizes
- Add accessibilityLabel to interactive elements

===================================================================
TROUBLESHOOTING
===================================================================

Blur not showing:
1. npm install expo-blur
2. npx expo start -c (clear cache)
3. Test on physical device

Performance issues:
1. Reduce BlurView intensity (40-50)
2. Limit number of BlurView components
3. Profile with React DevTools

Style issues:
1. Clear cache: npx expo start -c
2. Check Platform.select syntax
3. Verify imports from 'react-native'
